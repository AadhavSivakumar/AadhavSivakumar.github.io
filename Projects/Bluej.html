Bluej Project

This was a 3 week project that I worked on with Audrey Im. The objective of the project was to create our first java program that actively created a UI, instead of the being provided like with greenfoot and android studio. We needed to create a card game, that could be played single player or multiplayer with a computer. We decided to create solitaire, because the rules are pretty simple and one person can play it. Instead of using pictures for the cards, I used some jframe modules to draw rectangles with rounded corners to simulate cards. It plays similar to any other solitaire game, with a timer and a set of rules that pop up if you press a button.
