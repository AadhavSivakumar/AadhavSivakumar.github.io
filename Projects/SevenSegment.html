Bluej Project

This was a 3 week project that I worked on with Aaron Register, Roshni Vakil, and Antonia Leung. The purpose of the project was to create some logic to control 8 seven segment displays to create a message. Our message was the phrase "to infinity and beyond". In order to do this, we all divided the work into two segments each and wrote out the simplified logic for each one. Then, we combined it all into one big circuit, and cancelled out duplicated to reduce the number of gates. We ended up with 12 gates and 3 ICs, and we got the message to properly display.
